Projeto Destaque

CAT-Assist (2024)

Desenvolvido para modernizar e complementar os serviços de atendimento do CAT 166 (SESI), este projeto surgiu como solução para um problema real: a comunicação defasada entre a empresa e os usuários, que dependia majoritariamente de processos manuais, lentos e centralizados em poucas pessoas. A plataforma — composta por um site e um aplicativo — foi criada para ampliar e agilizar o acesso às informações, oferecendo uma experiência digital integrada e acessível aos usuários. O grande diferencial foi a implementação da Luna, um chatbot sendo um mascote de gata, desenvolvido com IA para oferecer atendimento automatizado 24/7. Além de tornar o suporte mais eficiente, a Luna trouxe um toque lúdico e acolhedor à interação com o público.

Node.js React React Native Firebase Google API

CAT-Assist (2024) - Mobile

O CAT-Assist Mobile é a versão otimizada para dispositivos móveis do sistema de atendimento do CAT 166 (SESI), desenvolvido em React Native para oferecer uma experiência ágil, acessível e moderna diretamente no celular do usuário. Com o mesmo poder de recursos da versão web, o aplicativo mobile traz todos os benefícios da Luna — a assistente virtual com personalidade de gata — para onde o usuário estiver.

Cotações (2024)

Atualizador de Cotações é um sistema de automação financeira desenvolvido em Python que realiza a coleta automática de cotações em tempo real do Dólar, Euro, Bitcoin e Libra através de API, armazenando os dados em planilha Excel com histórico completo. O projeto permite a atualização das cotações através de um botão no Excel utilizando macros VBA, oferecendo uma solução integrada para usuários que necessitam de acompanhamento de câmbio e criptomoedas de forma prática e eficiente, com possibilidade de conexão com Power BI para análise e visualização de dados.
